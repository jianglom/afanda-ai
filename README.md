阿-凡达是一个**有原则、懂时宜、会成长**的凡人AI框架。它不像那些全知全能的神，而像你我一样，从日常中学习，知道自己知道什么，更知道自己不知道什么。

## 📦 一分钟上手

```bash
pip install afanda-ai

from afanda import AFanDa

# 创建你的阿-凡达
ai = AFanDa(name="我的小凡")

# 开始对话
response = ai.think("冬天买凉鞋合适吗？")
print(response)
# 输出: 关于凉鞋，冬天热度只有8%，不太合时宜哦~
